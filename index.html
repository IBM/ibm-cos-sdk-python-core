Welcome to IBM COS Core Library for Python¶

ibm_botocore is a low-level interface to a growing number of S3 compatible Services. ibm_botocore is a key component in the ibm_boto3 project.

The ibm_botocore package is compatible with Python versions 2.6.5, Python 2.7.x, and Python 3.3.x and higher.
